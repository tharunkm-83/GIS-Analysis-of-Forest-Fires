# GIS-Analysis-of-Forest-Fires
A geospatial analysis of the severity and impact of Eaton Forest Fire, LA, which occured in Jan 2025. The analysis was performed in ArcGIS involving composite band creation, before and after - NBR and NDVI calculations and presenting the results in a proper format.
